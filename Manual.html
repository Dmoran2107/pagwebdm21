Instrucciones para utilizar la Pagina Web

Página de Inicio

  1. *Se muestra una pequeña serie de opciones
  2. *Al ingresar, verás una página con una imagen de fondo que cubre toda la pantalla
  3. *Puedes desplazarte hacia abajo para explorar las diferentes secciones.
  4. *Para poder utilizar el menú, se desplaza la flecha del mouse a la izquierda
  5. Recomendaciones Técnicas
  6. *Asegúrate de tener conexión a internet si las imágenes están enlazadas desde la web
  7. *Accede desde un navegador moderno (Chrome, Firefox, Edge, etc.) para mejor visualización
  8. *Si el diseño no se adapta bien a tu pantalla, reduce el zoom del navegador
